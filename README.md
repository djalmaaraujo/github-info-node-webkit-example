# Example APP using node-webkit for Lunch and Learn [at] Wobo


## To run
- node-webkit installed (https://github.com/rogerwang/node-webkit)
- git clone git@github.com:djalmaaraujo/github-info-node-webkit-example.git
- cd github-info-node-webkit-example
- npm install
- /path/to/your/node-webkit-app .
